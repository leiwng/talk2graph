# talk2graph — 产品需求

## 产品定位

为 K12（初中、高中）数学老师提供一个「用自然语言画数学图形」的 AI 工具。老师说画什么，AI 就画出精确的数学图形，并能基于已有图形进行持续修改。

## 目标用户

- 初中数学老师（几何图形：三角形、圆、多边形、辅助线）
- 高中数学老师（函数图像、解析几何、立体几何截面）
- 数学教辅内容创作者（出卷、做课件）

## 核心需求

### 1. 自然语言输入

用户用日常中文描述想要的图形，无需学习任何软件操作。

示例：
- 「画一个直角三角形，两条直角边分别是 3 和 4，标出边长」
- 「画 y = sin(x) 和 y = cos(x) 在 -2π 到 2π 的图像，用不同颜色区分」
- 「画一个圆，内接一个正六边形，标注圆心」

### 2. 精确的数学图形

- 长度比例准确，不畸变
- 角度标注精确
- 坐标系刻度正确
- 标注文字用中文

### 3. 持续修改

用户可以对已生成的图形提出修改要求，修改后未涉及的部分保持原样（位置、大小、粗细不变）。

示例：
- 「把直角标记从内侧移到外侧」
- 「去掉边长的标注，加一条斜边上的高」
- 「把坐标系的范围改成 -5 到 5」

### 4. 导出

- 下载为 SVG 矢量图（可插入 Word/PPT）
- 下载 TikZ 源代码（可插入 LaTeX 文档）

## 用户场景

1. **出卷子**：老师说「画一个底角 30° 的等腰三角形，腰长 5cm」，直接得到可放入试卷的精确图形，不再用 Word/GeoGebra 手动绘制。

2. **做课件**：老师说「画 y = x² 和 y = 2x + 1 的交点区域，用阴影标出来」，图形直接用于课堂演示。

3. **辅导学生**：老师边讲边改图——「加一条辅助线」「把这个角标出来」「把辅助线改成虚线」——交互式演示解题过程。

## 非功能需求

- 开箱即用，无需安装
- 国内网络可正常访问
- 首次上手时间 < 1 分钟
